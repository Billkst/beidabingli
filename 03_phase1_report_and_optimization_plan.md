# IgAN 激素疗效预测报告：第一阶段总结与优化方案

## 1. 任务概览
**目标**：基于277例 IgA 肾病患者的基线临床数据与 MEST-C 病理评分，预测激素治疗的短期（6个月，`label1`）与长期（12个月，`label2`）疗效。
**现状**：已完成第一轮“核心变量稳健版”建模，对比了逻辑回归（LR）与 XGBoost，以及是否纳入 MEST-C 评分的影响。

### 1.1 注释四点落实情况
1. **基线信息**：除 `Biopsydate` 外均作为基线信息使用，`Biopsydate` 已排除。
2. **共线性约束**：采用 `baseline GFR` 与 `MAP`，未与 `baseline Scr`、`SBP/DBP` 同时入模。
3. **缺失变量处理**：提供“核心稳健版”（低缺失）与“插补增强版”（高缺失+缺失指示变量）。
4. **MEST-C 对照**：同时训练“仅临床”与“临床+MEST-C”方案并对比性能。

---

## 2. 第一阶段结果总结 (Phase 1 Results)

### 2.1 核心指标对比
 **5折交叉验证** 得到的平均性能如下：

| 预测任务 | 特征方案 | 最佳模型 | AUC (区分度) | Brier Score (校准度) | 评价 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **短期 (label1)** | 仅临床 | XGBoost | 0.612 | 0.248 | 性能较弱，接近随机猜测 (0.5) |
| **短期 (label1)** | 临床 + MEST | XGBoost | 0.616 | 0.245 | 加入病理评分提升不明显 |
| **长期 (label2)** | 仅临床 | LR (ElasticNet) | 0.621 | 0.241 | 长期预测优于短期 |
| **长期 (label2)** | **临床 + MEST** | **LR (ElasticNet)** | **0.634** | **0.239** | **当前最佳**，病理信息有独立贡献 |

### 2.2 关键结论
1.  **“路遥知马力”**：模型预测长期疗效（12个月）的能力显著强于短期（6个月）。这符合临床逻辑——通过激素调节免疫反应进而改善蛋白尿和肾功能需要时间积累。
2.  **线性模型胜出**：在长期预测中，简单的逻辑回归（LR）表现优于复杂的树模型（XGBoost）。这提示在小样本（N=277）下，避免过拟合比捕捉复杂的非线性关系更重要。
3.  **S评分与M评分的重要性**：MEST-C 中，**S（节段硬化）** 呈现显著负相关（越硬化越难治），**M（系膜增生）** 呈现正相关（增生越活跃可能对激素反应越敏感）。

## 3. 结果图表解读指南

 `results/` 文件夹中，您会看到以下两类图表，解读方式如下：

### 3.1 校准曲线 (Calibration Curve)
*   **文件名**: `calib_lr_LongTerm_Clin+MEST.png` 等
*   **图示**: X轴是模型预测的概率（例如：模型说有80%可能有效），Y轴是实际观察到的有效比例（例如：这群人里实际有80%有效）。
*   **理想状态**: 点落在 **灰色虚线对角线** 上。
*   **如何看**:
    *   如果线在对角线 **下方**：说明模型 **高估** 了疗效（盲目乐观）。
    *   如果线在对角线 **上方**：说明模型 **低估** 了疗效（过于悲观）。
    *   **S型曲线**：两头准，中间不准，通常意味着模型区分度不够。

### 3.2 SHAP 特征重要性图 (SHAP Feature Importance)
*   **文件名**: `shap_summary_LongTerm_Clin+MEST.png` 等
*   **图示**: 横轴为 **Mean |SHAP value|**（平均绝对贡献），纵轴为特征名。
*   **如何看**:
    *   条形越长，代表该变量对模型预测的整体影响越大。
    *   该图强调“影响强弱”而非“影响方向”。方向性解释可结合逻辑回归系数或后续补充的依赖图。
*   **当前状态**: 已修复 SHAP 空白图问题，现输出为稳定的“条形图版本”，可在无图形界面环境中可靠显示。

## 4. 下一步：基于现有数据的性能提升方案 (Phase 2 Plan)

 AUC (0.63) 尚不足以支持临床应用。在不引入图像数据的前提下，我们有以下三个提升维度的策略：

### 策略 A: 挖掘“高缺失值”变量的潜力 (Data Rescue)
*   **问题**: 在第一阶段，我们丢弃了 `前驱感染`, `肉眼血尿`, `IgA`, `C3` 等变量，因为它们缺失率高（20%-50%）。
*   **机会**: IgA肾病是免疫性疾病，**IgA水平** 和 **C3补体** 可能包含核心的预后信息。**前驱感染** 更是鉴别 IgAN 发作诱因的关键。
*   **行动**:
    1.  使用 **KNN Imputer** 或 **MICE (多重插补)** 填补这些变量，而不是直接丢弃。
    2.  创建 **“缺失指示变量”** (例如 `IgA_missing`)，因为“未测量IgA”这件事本身可能就代表了某种临床亚型或病情轻重（非随机缺失）。

### 策略 B: 临床特征工程与交互 (Feature Impact)
*   **问题**: 目前只使用了原始变量。
*   **机会**: 医生判断病情往往看组合指标。
*   **行动**:
    1.  **构造比值**: `IgA / C3` 比值（常用于评估免疫活动度）。
    2.  **分箱 (Binning)**: 将 `Age` 分为 `<30`, `30-50`, `>50`，或将 `UTP` 分级，这可能比连续变量更能捕捉临床阈值效应。
    3.  **交互项**: `UTP * S评分`（蛋白尿高且伴有硬化的，预后可能加倍差）。

### 策略 C: 算法精细化 (Model Tuning)
*   **问题**: Phase 1 使用了默认或粗略的参数。
*   **行动**:
    1.  **GridSearchCV (网格搜索)**: 对逻辑回归的 `C` (正则化强度) 和 `l1_ratio` 进行细致搜索。
    2.  **SVM (支持向量机)**: 对于小样本高维数据，SVM 往往有奇效，值得尝试作为第三个对比模型。
    3.  **SMOTE**: 虽然使用了 class_weight='balanced'，但可以尝试合成少数类样本来进一步平衡数据。

**建议优先级**: 策略 A (找回免疫指标) > 策略 C (调参/换模型) > 策略 B (构造特征)。

## 5. 第二阶段初步结果（已执行策略A + 策略C）

### 5.1 关键变更
*   **策略A**：纳入高缺失变量（`前驱感染`, `肉眼血尿`, `IgA`, `C3`, `血尿（RBC）`, `Hb`），并通过KNN插补处理缺失。
*   **策略C**：对 LR 的 `C` 与 `l1_ratio` 网格搜索；新增 SVM 作为对照模型。
*   **特征增强**：自动加入 `IgA/C3` 比值与 `UTP × S` 交互项，并添加缺失指示变量。

### 5.2 性能对比（5折CV，进一步调参后）

| 预测任务 | 特征方案 | 最佳模型 | AUC (区分度) | Brier Score (校准度) | 备注 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **短期 (label1)** | Core | **XGBoost** | **0.696** | **0.242** | 提升显著 |
| **短期 (label1)** | Core+MEST | **XGBoost** | **0.706** | **0.221** | 当前短期最佳 |
| **长期 (label2)** | Core | **SVM** | **0.681** | **0.225** | 与XGB接近 |
| **长期 (label2)** | Core+MEST | **XGBoost** | **0.679** | **0.224** | 稳健提升 |

**结论**：加入 XGBoost 细致调参后，短期预测显著提升（最高 AUC≈0.706，且 Brier 降至≈0.221）；长期预测在 SVM 与 XGBoost 之间差距较小。综合可解释性与性能，建议长期主模型仍以 SVM/LR 报告为主，XGB 作为性能对照与补充。

## 6. 第三阶段深度优化结果（随机搜索 + 更细特征工程）

### 6.1 新增优化内容
*   **更细的随机搜索 (RandomizedSearchCV)**：对 LR/SVM/XGB 扩大搜索范围。
*   **特征增强**：加入年龄与UTP分箱（one-hot），增强非线性表达；保留缺失指示变量、IgA/C3比值与UTP×S交互项。

### 6.2 性能对比（5折CV）

| 预测任务 | 特征方案 | 最佳模型 | AUC (区分度) | Brier Score (校准度) | 备注 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **短期 (label1)** | Core | **XGBoost** | **0.699** | **0.220** | 短期最优稳定提升 |
| **短期 (label1)** | Core+MEST | **XGBoost** | **0.698** | **0.223** | 与Core接近 |
| **长期 (label2)** | Core | **SVM** | **0.680** | **0.228** | 与XGB接近 |
| **长期 (label2)** | Core+MEST | **XGBoost** | **0.685** | **0.223** | 当前长期最佳 |

**结论**：在更深度搜索后，短期预测 AUC 稳定在 **0.698~0.699**，长期预测达到 **AUC≈0.685**，Brier 也整体下降。整体最佳方案为：
*   **短期**：XGBoost（Core）
*   **长期**：XGBoost（Core+MEST）

## 7. 第四阶段：校准优化结果（Sigmoid / Isotonic）

### 7.1 校准策略说明
对第三阶段各任务的最佳模型进行概率校准，比较 **未校准** 与 **Platt（sigmoid）**、**isotonic** 的效果。

### 7.2 结果摘要（5折CV）

| 预测任务 | 最佳模型 | Base Brier | Sigmoid Brier | Isotonic Brier | 结论 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **短期 Core** | XGBoost | **0.2201** | 0.2232 | 0.2271 | 校准无益 |
| **短期 Core+MEST** | XGBoost | **0.2225** | 0.2275 | 0.2333 | 校准无益 |
| **长期 Core** | SVM | 0.2270 | **0.2253** | 0.2281 | Sigmoid 略有改善 |
| **长期 Core+MEST** | XGBoost | **0.2231** | 0.2281 | 0.2287 | 校准无益 |

**结论**：目前校准仅在“长期 Core + SVM”有轻微收益，其余任务校准反而降低校准度。建议默认保持未校准输出，仅在长期 SVM 方案中启用 sigmoid 校准。

## 8. 第五阶段：共线性对照试验（GFR vs Scr，MAP vs SBP/DBP）

### 8.1 试验设计
在保证“GFR 与 Scr 不同时入模、MAP 与 SBP/DBP 不同时入模”的约束下，构造 4 种特征组合：
1. GFR + MAP
2. GFR + SBP/DBP
3. Scr + MAP
4. Scr + SBP/DBP



### 8.2 结果摘要（Best AUC）
* **短期（label1）**：
  * 最优组合为 **GFR + MAP**（XGB AUC≈0.691）
  * 使用 **Scr 或 SBP/DBP** 会略降（最低≈0.656）

* **长期（label2）**：
  * **Scr + MAP**（含 MEST）可达到 **AUC≈0.686**（SVM）
  * **GFR + MAP** 也接近（AUC≈0.682）

**结论**：
1. 短期预测中，**GFR + MAP** 仍是最稳健方案。
2. 长期预测中，**Scr + MAP** 与 **GFR + MAP** 表现非常接近。
3. SBP/DBP 并未显著优于 MAP。
